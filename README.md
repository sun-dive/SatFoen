# SatFoen
A minimalistic stripped down optimized version of SVphone

The SatFoen revision of SVphone will lose the token creation section, however MPT will be utilized in the background in the form of a zero sat token for call initiation.

A MPT like token will also be used for software revisions and updates pushed to chain and verifiable by the SatFoen wallet via the Merkle proofs carried on a single sat that will trace back to it’s cryptographically provable original SatFoen NFT code.

If possible it will also include optimizations to enable single transaction call initiation.
